# Roadmap — Нарды Онлайн (React Native + Expo)

## Фаза 1 — Инфраструктура
- [x] Инициализация Expo проекта
- [x] Установка всех зависимостей
- [x] Настройка навигации (Native Stack: Login → Home → Game)
- [x] Подключение Supabase клиента (URL + anon key)
- [x] SQL миграция (таблицы `rooms`, `messages`, индексы, RLS, Realtime)

## Фаза 2 — Авторизация
- [x] Экран логина с вводом ника
- [x] Сохранение ника в AsyncStorage
- [x] Автологин при повторном запуске
- [x] Кнопка выхода (очистка storage, редирект на Login)

## Фаза 3 — Лобби (комнаты)
- [x] Создание комнаты (генерация 6-символьного кода)
- [x] Вход по коду приглашения
- [x] Список доступных комнат (FlatList + pull-to-refresh)
- [x] Автообновление списка каждые 5 секунд
- [x] Статус комнаты: «Ожидание» / «Игра»
- [x] Подсветка своих комнат

## Фаза 4 — Игровая логика нард
- [x] Доска 24 точки, массив int (+ = P1, − = P2)
- [x] Начальная расстановка
- [x] Бросок кубиков (2d6)
- [x] Дубли = 4 хода
- [x] Валидация ходов (canLandOn, isOwn, isOpponent)
- [x] Перемещение шашек с hit-механикой (сбитие на бар)
- [x] Вход с бара (P1 → точки 19-24, P2 → точки 1-6)
- [x] Bearing off (снятие, когда все в доме)
- [x] Автозавершение хода при отсутствии ходов
- [x] Определение победителя (15 снятых)

## Фаза 5 — UI доски
- [x] Компонент BackgammonBoard (треугольники, шашки, бар, зона снятия)
- [x] Выделение выбранной шашки (оранжевый бордер + glow)
- [x] Подсветка валидных ходов (жёлтый)
- [x] Кубики с реальными точками (DieFace), затемнение использованных
- [x] Кнопка «Бросить кубики» / отображение результата
- [x] Счётчик снятых и бара для обоих игроков

## Фаза 6 — Мультиплеер (Supabase Realtime)
- [x] Синхронизация game_state через UPDATE на таблице `rooms`
- [x] Подписка на `postgres_changes` (UPDATE rooms, INSERT messages)
- [x] Сброс выделения при получении чужого хода
- [x] Блокировка ввода на ход соперника (isMyTurn)

## Фаза 7 — Чат
- [x] Компонент Chat с FlatList
- [x] Отправка через Supabase insert
- [x] Realtime подписка на новые сообщения
- [x] Свои/чужие сообщения разным цветом
- [x] Переключение доска ↔ чат кнопкой

## Фаза 8 — Стилизация
- [x] `twrnc` (Tailwind для React Native)
- [x] Тёмная тема (gray-900)
- [x] Amber акценты (кнопки, активный игрок)
- [x] Деревянная текстура доски (коричневые оттенки)
